# wCDM vs CPL Dark Energy Model Comparison

This repository contains code to compare wCDM and CPL dark energy
models using nested sampling (Dynesty) on supernova luminosity
distance data.

The analysis includes:

- `model_comparison.ipynb` — Notebook performing Bayesian model
comparison and generating corner and trace plots.
- `data_2.txt` — Supernova luminosity distance dataset used in the notebook.
